Epicodus Track Suggester

Epicodus has consistently been recognized as one of the top vocational bootcamps for aspiring developers in America, They offer multiple programs in numerous languages the most beneficial of which are their educational tracks. Although similar, each track is fairly unique in it's execution. It can be rather overwhelming to navigate.

Currently each track consists of:

  • An Intro to Programming course
  • An intermediate JavaScript course
  • An infield internship
  • And two 5 week deep-dive of one of the following pairs
    1. Ruby and Rails
    2. PHP and Drupal
    3. Java and Android
    4. CSS and Design
    5. C# and .NET
